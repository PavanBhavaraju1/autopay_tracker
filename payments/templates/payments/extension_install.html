Install AutoPay Extension

Step 1 – Download the extension ZIP

⬇️ Download AutoPay Extension (.zip)

Step 2 – Load it in Chrome

  1. Open Google Chrome.
  2. Go to chrome://extensions in the address bar.
  3. Turn on Developer mode (top‑right corner).
  4. Click “Load unpacked”.
  5. Unzip the downloaded file and select the unzipped autopay_extension folder.

Step 3 – Use it on checkout pages

  • Visit a checkout page (Netflix, Amazon, Spotify, etc.).
  • Click the AutoPay extension icon.
  • Save the card or subscription to send it to your AutoPay Tracker dashboard.
